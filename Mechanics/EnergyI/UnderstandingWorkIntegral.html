What the question looks like:
The function:

F(x) = nx


Represents the force applied to an object, where x represents the distance travelled. (df - di)
Calculate the work done to the object

SOLUTION:
To understand the solution, we first must explain the SI units of force:

F = Kg m/s2


Where:
Kg is Mass (kilograms)
m is distance (meters)
s is the time (seconds)

By integrating with respect to the distance (m), we learn that:

∫F dm = ∫Kg m/s2 dm


Taking out the constants:

∫F dm = Kg/s2∫m dm


We are left with the indefinite integral

∫F dm = [Kg m2/2s2] + c


This integration is the same as the calculation of SI units for work:

W = FD
W = Kg m2/2s2


To solve this problem, you will take the defenite integral (from di to df) For this problem in particular, my solution would be:
